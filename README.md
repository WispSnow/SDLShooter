# SDLShooter
**SDLSHOOTER** is a cross-platform space fighter game developed in C++ with SDL2 (SDL3).

> This project is a teaching demonstration project; it is the 1st episode in a series of tutorials titled "[C++ 游戏开发之旅](https://cppgamedev.top/)".

## Control
```
W,A,S,D - to move;
J - to attack;
F4 - toggle fullscreen;
ESC - back to title
```
