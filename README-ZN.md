# SDLShooter

**SDLSHOOTER** 是一个使用 C++ 和 SDL2 (SDL3) 开发的跨平台太空射击游戏。

> 本项目是一个教学演示项目，是系列教程 “[C++ 游戏开发之旅](https://cppgamedev.top/)” 的第一章。

## 操作说明
```
W,A,S,D - 移动
J - 攻击
F4 - 切换全屏
ESC - 返回标题
```
